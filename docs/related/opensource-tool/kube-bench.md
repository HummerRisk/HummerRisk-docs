#### 一、Kube-bench 是什么

!!! abstract "Kube-bench 是什么"
    1. kube-bench 是一种工具，可通过运行CIS Kubernetes Benchmark中记录的检查来检查 Kubernetes 是否已安全部署。
    2. 检查 Kubernetes 是否根据 CIS Kubernetes 基准中定义的安全最佳实践进行部署。
