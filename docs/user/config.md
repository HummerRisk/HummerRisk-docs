### K8s 部署检测

云原生部署检测功能，输入 K8s 部署配置 YAML 文件，即可进行部署检测，输出部署配置检测结果。

> K8s 部署检测结果列表，点击"统计按钮"进入详情列表，点击"状态按钮"查看日志与报告。
