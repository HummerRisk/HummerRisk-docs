### 云资源合规报告

!!! abstract "合规报告列表"
    - 合规报告页面，展示按规则组与等保条例统计资源合规报告信息。
    - 云资源检测合规报告，按已检测规则组分类，可查看/下载云资源合规报告。

![合规报告](../img/release/0.5.0/img.png){ width="95%" }

!!! tip "下载云账号整体合规报告"
