## 文件检测

!!! info "文件检测"
    手动添加项目依赖文件(单独文件或包含依赖文件的压缩包zip或tar)，例如：pom.xml、package.json、yarn.lock、pnpm.yaml等，检测依赖项的漏洞风险等信息。
